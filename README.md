# Exercícios de Estrutura de Repetição - C# (.NET)

Este repositório contém resoluções de uma lista de exercícios de lógica de programação focada em **Estruturas de Repetição** e **Validação de Dados** utilizando C#.

## 🚀 Tecnologias e Conceitos Utilizados

* **Linguagem:** C# (.NET 10.0)
* **Estruturas de Repetição:** `while` e `for` para controle de fluxos e repetições baseadas em condições.
* **Validação Robustas:** Uso de `int.TryParse` e `double.TryParse` para garantir que o programa não quebre (crash) caso o usuário digite dados inválidos (como letras em campos numéricos).
* **LINQ (Language Integrated Query):** Manipulação de coleções (`List<int>`) de forma elegante através de métodos como `.Sum()`, `.Max()` e `.Average()`.
